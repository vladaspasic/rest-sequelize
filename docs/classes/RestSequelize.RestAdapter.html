    RestSequelize.RestAdapter

    Extends RestSequelize.Object
    Module: rest-sequelize

    Adapter for defining RESTFull schema for Models. Should handle all CRUD operation runned against a Model.

    All incoming payloads are normalized, and the serialized into a response object ready for flushing.
