    RestSequelize.Resolver

    Extends RestSequelize.Object
    Module: rest-sequelize

    Resolver used by the RestAdapter to find Models, RestSequelize.RestService, RestSequelize.Serializer and RestSequelize.Deserializer classes.

    Uses caching to store and resolve already looked up modules.
